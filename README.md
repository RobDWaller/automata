# Automata

An Elementary Cellular Automata library for PHP. All credit goes to Stephen Wolfram and Melanie Mitchell.

## Rules

```
000 = 0
001 = 1
010 = 1
011 = 1
100 = 0
101 = 1
110 = 1
111 = 0
```
